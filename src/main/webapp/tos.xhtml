Privacy notice

RePol collects usage information about you when you visit the website through common web server logs. The information that is automatically collected and stored includes:

  • the Internet domain and your IP address;
  • the type of browser and operating system;
  • date and time;
  • pages visited.

This is only used internally for technical troubleshooting, to improve the features of RePol, and to record aggregate statistics and will never be shared with third parties.

The information you enter in RePol will not be stored and will be lost when closing the corresponding web browser session. If you wish, you may download your input data. By using this application, you agree to the manner and purposes of data processing.

RePol uses the following cookies:

You can disable cookies in your browser settings. However, this might limit the functionality of this application.
